2019 SIGKDD - Conventional Tutorial

Description: With the prevalence of networked systems, such as social media and protein-protein interaction networks, learning from networks has become a crucial task in practice. It is well recognized that network data is sophisticated and large-scale. In this tutorial, we systematically introduce learning from networks algorithms, along with the theory and applications. After introducing the history and formally defining the problems, following the trend, we roughly categorize the learning algorithms into four classes, i.e., feature selection based, network embedding based, attributed network embedding based, and graph neural based. Theoretical analysis and real-world applications are given for each class. We conclude with open problems and recent trends.
